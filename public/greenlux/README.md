# GreenLux Public Image Assets

All images in this folder were downloaded from GreenLux Residency's own WordPress uploads at `https://greenluxresidency.com/wp-content/uploads/...`.

The public site does not hotlink Airbnb, Booking.com, Skyscanner, or Unsplash images.

## Notes

- `property/` contains clean GreenLux exterior, terrace, and lounge imagery used for hero and trust sections.
- `rooms/` contains curated room/unit galleries. Each gallery is ordered as lead room image, alternate angle, bathroom, kitchen/lounge or shared-space image, then detail shots.
- Galleries intentionally use only 5-8 selected images per unit, avoiding the darkest, most cluttered, or duplicate-angle WordPress uploads.
- Room 9 and Room 10 are mapped from the current GreenLux WordPress apartment posts exposed by the WordPress API.
- No image downloads failed during this implementation pass.
